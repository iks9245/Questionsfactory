        Project

        用問題驅動思考，而不是只追逐答案

        Questionsfactory 是一個單頁訓練工具，目標是幫使用者建立「判斷問題品質」的能力。它透過辨識、解析、改寫、AI 教練回饋與個人化能力模型，訓練使用者看見目的、情境、邊界、深度、行動與假設。

        辨識好問題 從多個相似選項中判斷哪個問題更能推動思考。
        升級弱問題 把空泛、封閉或失焦的問題改寫成可行動的問題。
        AI 問題教練 支援雲端與本機模型，評改你的問題並提出追問。

        AI 出題

        用 LLM 產出新問題

        選擇供應商、填入自己的 API key，產生一組新的辨識題與改寫題。

        預設一次產生 10 題，最多 30 題。品質引擎會先要求模型多產生候選題，再檢查選項長度、最佳答案是否過於明顯、評分理由是否完整；可修正的題會本地校準，不合格的題會自動重試補足。雲端 API key 會從這個瀏覽器直接送到你選擇的供應商。本機端點可填 base URL，例如 http://127.0.0.1:11123/v1，頁面會自動補成 /chat/completions；若瀏覽器擋住 localhost 請求，通常需要在本機模型服務開啟 CORS。若勾選記住，key 會以純文字存在這台瀏覽器的 localStorage。
